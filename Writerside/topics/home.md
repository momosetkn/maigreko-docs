# %product%

This is a minimal Writerside setup sample.

This project follows the general structure of `liquibase-kotlin-docs`, while keeping only the files needed for an initial setup.

## Included

- `writerside.cfg`
- One instance definition: `maigreko.tree`
- Variable file: `v.list`
- Two sample pages

## Next steps

1. Install the Writerside plugin in IntelliJ IDEA
2. Open this repository
3. Open `Writerside/maigreko.tree` or `Writerside/topics/home.md` and preview the docs

[Go to the sample page](sample-page.md)

# maigreko-docs

Minimal Writerside setup for this repository.

## Structure

The documentation module lives under `Writerside/`.

## Open locally

1. Open this repository with IntelliJ IDEA
2. Install the Writerside plugin if it is not installed
3. Open `Writerside/maigreko.tree` or `Writerside/topics/home.md`
4. Use the Writerside preview/build actions from the IDE
